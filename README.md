# About my DVD collection sorter
A script that chooses a random DVD from my collection based on user input on genre and runtime.
Genres are animated, adventure, comedy, drama, fantasy, musical, scifi and superhero.
Runtimes are under 90 minutes; greater or equal to 90 minutes; greater or equal to 120 minutes and greater or equal to 180 minutes.
It chooses from a total of 32 DVDs, accounting for genre overlaps.
